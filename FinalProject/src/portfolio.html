Pixel Art Drawing App Live Chat / Dating App Wheel of Fortune Game Replica Micro Blog App 3D Interactive Rubik's Cube
This application allows a user to pick a color and draw pixel art! The user also has the ability to save their current artwork and load previously saved work. Lastly, there is also a button that allows the user to erase the current drawing. This is a mock dating app. The user can create a profile to: view other users, access an inbox that contains a list of current msg threads, live chat with other users, view connections, manage their profile page and set a profile picture. This is a wheel of fortune game replica I created to announce the name of my 2nd son. I built this app using React. It was played at my wife's baby shower. It was a lot of fun to play with our family and reveal his name. The user simply clicks on the squares to reveal the letter. I created this project as a tribute to a fellow Soldier who lost his life. SPC Ritter was a Soldier is the U.S. Army, a classmate and my friend. A user can create an account, view all posts, create a post, view posts made by the current user, and toggle a dark/light theme. I built this Rubick's cube app to play around with the Javascript Three.js library. I learned a lot about this library and enjoyed working on this small project. The user can rotate the cube by clicking and dragging and spin a face of the cube using specific keyboard buttons.
