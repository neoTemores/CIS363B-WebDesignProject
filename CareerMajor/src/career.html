Desired position: Front End Software Engineer

Duties: Front end developers are usually responsible for creating visual elements and/or user interfaces (UI) of a web application. The main focus is making sure the user has a good experience, the application runs smoothly, that it is visually appealing and easy to use.

They are usually required to have a lot of knowledge in various topics that include but not limited to:

  • HTML: must be familiar with syntax and proper webpage structure.
  • CSS: ability to style pages and/or know how to pull in styling libraires/frameworks.
  • JavaScript: should know vanilla JS very well and be familiar with core programming and logic concepts. Also helps to know how to work with popular JS frameworks like React.
  • Responsive design: must understand the importance of dynamic websites so that the presentation is not negatively impacted by the user's device size.
  • Version control: should be familiar with modern version control practicies and workflows. Git is a popular version control tool that all developers should know.
  • Application performance: understand how to optimize application performance to minimize load times and resources.
  • Testing: how to properly write unit tests and debug code.

Expected growth: According to the Bureau of Labor Statistics the expected job growth for software developers is 26% over the next 10 years. This is considered a much faster than average job outlook.

Salary: According to the Bureau of Labor Statistics the average salary for a software engineer is $127,260 depending on experience and location.

Working conditions: The job of a software engineer is usally a full time position, and requires sitting in front of a computer for extended periods of time, being indoors, attending meetings and working as part of a development team.

Major companies: It would be nearly impossible to find a company in todays economy that does not have an online presense. So its fair to say that nearly every company in the world has a need for a software engineer.
