Welcome to Neo's week 5 CIS363B project!

The goal of this weeks' project is to create a website the provides information about my major and future career oportunites. We also covered an introduction to JavaScript this week and must incorporate a few JS functions in our project. One function will display the current date in the footer of the page.

Please use the navbar above to navigate to the "Major" page to see information about my degree and to the "Career" page to see potential career opportunites.

I used a few functions in this project:

  • setCurrentDate() is the main function that handles setting the date in the footer.
  • parseDayOfWeek(day) returns the day of the week as a string.
  • parseDateSuffix(date) returns the correct date suffix (st, nd, rd, th).
  • parseMonth(month) returns the month as a string.
  • updateNavbar() updates the highlighted navbar link.
