FAQ's about this web app 🤔


Who created it?

Neo's Emoji Playground was built by Neo Temores as part of a Web Inteface Design (CIS-363B) class project for DeVry University.


What does this app do?

This application has 3 main functions:

  • View all
    • The view all page allows a user to click through all of the emoji's that are available via the free API.
  • Category / Group
    • The Category/Grup page allows a user to filter emoji's based on category or group.
  • Test Unicode
    • The Test Unicode page will allow a user to click a button to retreive a random emjoi card. This page also allows a user to enter an Emoji Unicode into a text box and click a button to view it.

Why build this app?

This application provides an interactive way to look at various kinds of emojis and their respective unicode. You can paginate through every emoji available from the free API, filter by category or group and even enter unicodes to view.


How was this site created?

This site was created using HTML5, CSS3 and Vanilla Javascript. No libraries or frameworks were used to create this site. The emojis are pulled from a free emoji API using Javascript fetch and async/await.
