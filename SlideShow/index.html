Welcome to Neo's CSS Slide Show!

Hello, my name is Neo Temores and I created this page as a project for my CIS363B Web Design class at DeVry University. I decided to build the slideshow myself instead of copying code online. However, I did read the article A Pure CSS3 Cycling Slideshow to understand more about CSS animations.

Here is some info about this project:
  • The slideshow is done in 100% CSS.
  • The animation is 25s long and cycles though 5 images.
  • Each image annimation has a different delay applied, increasing by intervals of 5s.
  • The slides "move" by animating the opacity of each image.
  • The loading bar container hides the overflow of the actual progress bar.
  • The progress bar is animated to move from left to right and change from light green to dark green.
  • When the slideshow container is hovered, I pause both the image and progress bar animations using the animation-play-state property.
  • All of the images are public domain and were downloaded from pexels.com.
